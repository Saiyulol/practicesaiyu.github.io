Introduction à HTML et CSS

HTML

HTML (HyperText Markup Language) est utilisé pour baliser un contenu, c'est à dire pour le structurer et lui donner du sens.
Le HTML sert, entre autres choses, à indiquer aux navigateurs quel texte doit être considéré comme un paragraphe, quel texte doit être considéré comme un titre, que tel contenu est une image ou une vidéo.

1. Les bases du HTML

1-1. Éléments, balises et attributs

Il y a trois termes dont vous devez absolument comprendre le sens en HTML. Ce sont les termes élément, balise et attribut.

Les éléments (ou element en anglais), tout d'abord, vont nous servir à définir des objets dans notre page.

<a href="URL">Hello World</a>

Les éléments sont constitués de balises. Dans la majorité des cas, un élément est constitué d'une paire de balises : une balise ouvrante et une balise fermante.

<p>___________</p>

Liste non exhaustive de balises HTML

Structure du document
Balise Description
<!DOCTYPE> Définit le type de document et la version de HTML utilisée.
<html> Élément racine qui enveloppe tout le contenu de la page HTML.
<head> Contient les métadonnées de la page, comme le titre et les liens vers les fichiers CSS et JavaScript.
<body> Contient le contenu visible de la page, comme les textes, images et autres éléments.
Informations de la page
Balise Description
<title> Définit le titre de la page affiché dans l'onglet du navigateur.
<meta /> Fournit des métadonnées sur le document, telles que les descriptions, les mots-clés, et les paramètres de l'auteur.
<base /> Spécifie l'URL de base pour toutes les URL relatives dans le document.
<link /> Utilisé pour lier des ressources externes comme des feuilles de style CSS.
<style> Contient des règles CSS pour la mise en forme du document.
<script> Inclut du code JavaScript ou référence un fichier JavaScript externe.

Balise Description
<div> Section de division
<span> Section en ligne
<section> Section
<nav> Navigation
<header> En-tête
<footer> Pied de page
<article> Article
<aside> Contenu associé

CSS

CSS (Cascading Style Sheets) est utilisé pour appliquer des styles à un contenu, c'est-à-dire à le mettre en forme. Ainsi, avec le CSS, on pourra changer la couleur ou la taille d'un texte, positionner tel contenu à tel endroit de notre page web ou ajouter des bordures ou des ombres autour d'un contenu.
